# Boston House Price Prediction

An end-to-end Machine Learning project that predicts Boston house prices using regression models and exposes predictions through a FastAPI REST API.

## Features
- Data preprocessing pipeline
- Feature engineering
- Model training using XGBoost
- FastAPI backend
- Docker support
- CI/CD using GitHub Actions
- Deployment on Render

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- FastAPI
- Docker
- GitHub Actions

## Dataset
Boston Housing Dataset containing housing-related features such as:
- RM
- LSTAT
- PTRATIO
- TAX
- CRIM

Target:
- MEDV (Median House Price)

## Model Performance
-RandomForestRegressor-R2 score-69%
